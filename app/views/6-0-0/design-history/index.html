Iterated content for Start, Login and Task page

This version saw some considerable changes, due to some great work by a Content Designer who recently joined the team. Link to content notes

Iterated content for download journey

The content work also covered the download journey.

An initial rewording of the Service name was introduce, however a full workshop will be needed.

Changed download method

Previously, the CSV file would automatically download once a user reached the confirmation screen. This is has now changed to a manual download button, this is an accessibililty improvement, and due to the MVP design not using Javascript.

Changed download options available for MVP

'Users whose GOV.UK One Login may have been compromised' was removed as a download option for MVP, as scope for MVP focuses on user ID subjects only, with device and credential concerns to be added as MVP+ functionality.

Added help links

Content was added on all screens to provide users a channel for support. For MVP, the link will send an email to the UCD team, so that UCD can triage all contact from users in private beta.

Added feeback link

A Beta phase banner was added, to provide users with a channel for their feedback about the service. For MVP, the link will send an email to the UCD team, so that UCD can triage all contact from users in private beta.

Removed stand alone support page

Due to the MVP approach to support channels, the need to a stand alone support screen was reviewed, and instead a link to a shared email address was added.

Updated flow for the add journey, to factor in device concerns

Post MVP fucntionality includes allowing for users to add information that relates to other user attributes, which include IP address, email addresses etc.

Updated upload CSV for post MVP

Post MVP fucntionality includes allowing for users to add information by a CSV. The initial design upon reflection seems very bloated, and due to changes in retention poilcy and startegic ambition of the project this new light touch approach is worth exploring.

To see this option you should select "Add by CSV" on the setup screen.
