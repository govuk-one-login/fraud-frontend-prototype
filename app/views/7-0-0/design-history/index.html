Background

The password reset journey has been designed for SSF UI users to reset their password, using a ‘I have forgotten my password’ link on the sign in screen. The designs created match the existing password reset pattern in the public facing screens of GOV UK One Login.

Impoved security for mfa codes and sign in

New unhappy path screens have been introduced if a user has entered sign in details, an MFA or OTP, too many times, which apply a temporary block onto the account for 15 minutes - to align to GOV.UK One Login public facing
